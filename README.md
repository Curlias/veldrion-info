## Cómo usar

1. Abrir `index.html` en cualquier navegador (doble clic o `open index.html` en terminal).
2. Personaliza los enlaces en `index.html` (atributos href) con URLs específicas si es necesario.
3. Sustituye la imagen del avatar y el `logo.svg` por versiones oficiales de Veldrion.

## Despliegue rápido

### GitHub Pages
1. Sube estos archivos a un repositorio de GitHub.
2. Ve a **Settings → Pages** y habilita Pages desde la rama `main` (carpeta root).
3. Tu página estará en `https://[usuario].github.io/[repositorio]/`

### Netlify / Vercel
1. Conecta el repositorio o arrastra la carpeta del proyecto.
2. Deploy automático en segundos.

### Hosting tradicional (S3, Cloudflare Pages, etc.)
1. Sube los archivos a tu bucket/hosting.
2. Configura el dominio personalizado si lo deseas.

## Personalización

- **Colores:** Variables CSS en `:root` del archivo `styles.css`.
- **Tipografía:** Cambia el enlace de Google Fonts en `<head>` de `index.html`.
- **Contenido:** Modifica textos, bio y enlaces directamente en `index.html`.
